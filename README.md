# HOUSE OF MUSCLE - Landing Iscrizione (Prototype)

Landing one-page in React + TypeScript per iscrizione abbonamenti palestra.

## Requisiti

- Node.js 20+
- npm 10+

## Avvio locale

```bash
npm install
npm run dev
```

Apri il link mostrato in console (tipicamente `http://localhost:5173`).

## Build produzione

```bash
npm run build
npm run preview
```
